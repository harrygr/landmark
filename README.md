# Landmark

A geospatial analysis library for Elixir.

Landmark is designed to work with structs provided by the [Geo][geo] library

## Installation

Add `landmark` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:landmark, "~> 0.3.0"}
  ]
end
```

[geo]: https://github.com/felt/geo
